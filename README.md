# IRCTC-ticketbooking-automation
This is implemented by using selenium through java.

AIM: To select train tickets for the specified departure and destination railway station after four days from the present day for sleeper class for a physically challenged
     person.

Workflow:
Upon the rendering of Results component, the screen shot will be taken only for the trains that have the same departure and destination cities specified.
payment for the train selected is not implemented as of now!
Multi threading is utilized to run the application on multiple browser instances simultaneously after specifying the total no.of threads along with specifying which browser to - run by each thread(Chrome and edge are being used to do so as of now).
